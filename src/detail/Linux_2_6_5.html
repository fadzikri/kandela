  • Linux_2_6_5
Last updated at 2019-05-16 17:06:16

2.6.5 was released on April 4, 2004. changelog

  • Adaptive lazy readahead commit

  • Netpoll infrastructure (LWN article). commit

  • Hotplug CPUs: Sysfs Online Attribute commit

  • CRYPTO: Add Michael MIC algorithm commit

  • NET: Add packet delay scheduler commit

  • New "kref" reference counting mechanism (LWN article). commit

  • vm: per-zone vmscat stats commit

  • reiserfs: AIO support commit

  • selinux: Conditional policy extension and MLS detection support commit

  • Add SCSI lots of disk support commit

  • Add SCSI transport attributes commit

  • Add boot parameters acpi_osi and acpi_serialize. acpi_osi will disable the _OSI method, which tells the BIOS to behave as if Windows is the OS. acpi_serialize is for debugging AE_ALREADY_EXISTS failures commit

  • read-only support for UFS2 commit

  • Drivers & architectures

    • ppc64: iSeries virtual tape driver commit

    • ppc64: Add numa=off command line option commit

    • ppc64: allow MAP_FIXED hugepage mappings commit

    • ALPHA: Detect and export cache shapes to userland. commit

    • sh: hugetlb support commit

    • sh: DMA Mapping API commit

    • qlogicfas: support multiple cards commit

    • USB: Altsetting/interface update for USB image drivers commit

    • USB: usbnet and ALI M5632 commit

    • libata: Add new driver for Vitesse VSC-7174 commit

    • Add DEC LK201/LK401 keyboard support commit

    • input: Add driver for DEC VSXXX mice commit

    • USB: visor patch for Samsung SPH-i500 commit

    • USB: add new USB Touchscreen Driver commit

    • USB: add driver for ATI USB/RF remotes commit

    • USB: add IRTrans support to ftdi_sio driver commit

    • I2C: add w83627hf driver commit

    • Add new Prism54 wireless driver commit

    • Add Promise SX8 (carmel) block driver commit

    • Add adapter support to aacraid driver commit

    • Added the au88x0 drivers for Aureal soundcards commit

    • Added snd-atiixp driver for the ATI IXP150/200/250 AC97 controllers commit

    • Add Intel-compatible onboard MC97 modem driver commit

    • I2C: New chip driver: ds1621 commit

    • Intel8x0: add Intel ICH6 and ESB commit

    • cdrom: allow random write to cdrom devices with profile 2 (removable disk) commit

    • ATI IXP IDE support commit

    • AGPv3 generic support. If a chipset doesn't need anything funky, then it can use these routines commit

    • Bluetooth: Add support for AVM BlueFRITZ! USB v2.0 commit

    • sh: DAC ODD driver commit

    • USB audio generic driver: high speed support commit

    • CDROMREADAUDIO dma support commit
