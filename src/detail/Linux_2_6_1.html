    • FAT: add readv/writev support commit

    • reiserfs: commit_max_age mount option commit

    • USB Lego Infrared Tower support and other things commit

    • dvb: Add new dvb bt8xx driver commit

    • Intel 440gx PCI IDs commit

    • Support centrino 1GHz commit

    • Add support for SGI's IOC4 chipset commit
